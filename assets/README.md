# Branding assets

Drop the MAIC LAB logo here to brand the whole app (top bar + sidebar).

The app picks up the **first** file that exists, in this order:

- `maic_logo.png` ← recommended (wide wordmark, transparent background)
- `maic_logo.svg`
- `maic_logo.jpg` / `maic_logo.jpeg`
- `logo.png` / `logo.svg`

Optional small square mark (used for the collapsed sidebar and the browser tab):

- `maic_icon.png` (or `maic_mark.png`)

Recommended sizes:

- Wordmark: ~ 240–400 px wide, height ~ 60–80 px, transparent PNG.
- Square mark: 64×64 or 128×128 px.

No file here → the app falls back to a plain "MAIC LAB" text wordmark and the ✅
tab icon, so nothing breaks. Add a file and redeploy to see the logo.
